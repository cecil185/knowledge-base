## Overview

Simon Willison's link post highlighting Max Woolf's detailed exploration of AI agent coding capabilities. Willison endorses Woolf's progression from skeptic to cautious believer and adds his own perspective on the current state of coding agents.

## Key Points

### Woolf's Progression
Willison highlights Max Woolf's journey through increasingly ambitious projects using Claude Opus 4.5 and Codex, culminating in `rustlearn` — a Rust implementation of ML algorithms comparable to scikit-learn. The projects serve as open-source evidence that modern agents produce real, verifiable output.

### Model Quality Leap
Woolf's core claim, endorsed by Willison: models released after November 2025 (particularly Opus 4.5) represent an order-of-magnitude improvement over predecessors. The difficulty is communicating this without appearing to engage in hype.

### Willison's Own Experience
Willison references using Claude Code to build a Rust word cloud CLI tool, corroborating Woolf's findings about agent capabilities in systems programming tasks that would traditionally require deep language expertise.
